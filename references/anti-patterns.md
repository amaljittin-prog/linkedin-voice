# Anti-Patterns — What Kills Your LinkedIn Reach

## Why This File Exists
The LinkedIn algorithm actively detects and demotes content that matches these patterns.
Around mid-2025, the feed became saturated with AI-generated content that all started sounding identical.
Users stopped engaging. Algorithm started burying it. AI-sounding content now gets 47% less reach.

Read this before reviewing any post output.

---

## PHRASES THAT INSTANTLY SIGNAL AI / CORPORATE VOICE

Never use these. Ever.

**The Excited Announcer:**
- "I am excited to share..."
- "Thrilled to announce..."
- "I'm delighted to..."
- "Honored to be..."
- "Humbled and grateful..."

**The Corporate Buzzword:**
- "Game-changer" / "game-changing"
- "Leverage synergies"
- "Circle back"
- "Bandwidth"
- "Move the needle"
- "Low-hanging fruit"
- "At the end of the day"
- "Think outside the box"
- "Best practices"
- "Value-add"

**The Vague Opener:**
- "In today's fast-paced world..."
- "In this day and age..."
- "As we navigate the ever-changing landscape..."
- "The world of [X] is constantly evolving..."
- "Now more than ever..."

**The AI Giveaway:**
- "Delve into"
- "It's important to note that"
- "I hope this finds you well"
- "In conclusion"
- "To summarize"
- Any numbered list that starts: "1. First... 2. Second... 3. Finally..."

## FORMAT PATTERNS THAT KILL REACH

**Wall of text:**
```
❌ BAD:
I've been thinking a lot about my internship experience this summer and wanted to share some thoughts. 
When I first started, I was really nervous and didn't know what to expect. The first week was hard 
and I made a lot of mistakes. But over time I learned so many things that I think will be valuable 
for my future career. Here are some of the lessons I took away from this experience...
```

```
✅ GOOD:
My first week at the internship, I made 3 mistakes.

Not small ones. Real ones.

The kind where you have to go to your manager and say "I messed up."

Here's what I learned from each one — and what I'd do differently.
```

**Too many hashtags:**
```
❌ BAD:
#BBA #Student #India #LinkedIn #Career #Internship #Learning #Growth #Success 
#Professional #Network #Jobs #Placement #MBA #Business #Startup #Entrepreneur
```

```
✅ GOOD:
#BBAStudent #CareerAdvice #IndianStudents
```
(Max 3. Always at the very end. Never mid-post.)

**Link in the post body:**
```
❌ BAD:
"I built this tool — check it out here: github.com/username/linkedin-voice
It's free and you can download it now..."
```

```
✅ GOOD:
[Post body — no links]

[FIRST COMMENT]: "For those who asked — the free skill is here: github.com/username/linkedin-voice"
```

**The generic closing question:**
```
❌ BAD:
"What do you think? Let me know in the comments!"
"Thoughts? Would love to hear from you!"
"What's your opinion on this?"
```

```
✅ GOOD:
"What's one thing your college didn't teach you that you had to figure out yourself?"
"Has this ever happened to you — tell me the specific moment."
"What would you have done in my situation? Genuinely curious."
```

## POST STRUCTURES THAT BACKFIRE

**The Resume Post:**
```
❌ NEVER DO THIS:
I am a BBA student at [college] with experience in marketing, finance, and operations.
I have completed internships at [X] and [Y]. I am passionate about entrepreneurship 
and hope to build a career in consulting. Open to opportunities!
```
This gets zero reach because it has zero value for anyone else.

**The Motivational Poster:**
```
❌ NEVER DO THIS:
"Success is not a destination, it's a journey.
Work hard. Stay focused. Never give up.
Your dreams are valid. 💪
#Motivation #Success #Hustle"
```
Generic inspiration with no specific story = no engagement.

**The Humble Brag:**
```
❌ NEVER DO THIS:
"Excited to share that I received an offer from [company]!
It's been a tough journey but hard work pays off.
Grateful for this opportunity! 🙏"
```
This generates likes from family members, not real reach.

**The Over-Tagged Post:**
```
❌ NEVER DO THIS:
Tagging random well-known people to get their attention.
"@[Famous person] what do you think about this?"
```
This looks desperate and LinkedIn's algorithm ignores forced tagging.

---

## THE DEATH SENTENCE COMBOS
These combinations guarantee a post fails:

**Combo 1:** Generic opener + wall of text + no question at end
**Combo 2:** Humble brag + corporate buzzwords + 10 hashtags
**Combo 3:** Motivational quote + nothing personal + "tag someone who needs this"
**Combo 4:** Job announcement + formal language + link in body
**Combo 5:** List of achievements + no story + no specific insight

---

## THE AUTHENTICITY TEST
Before finalizing any post, ask:

1. Could a stranger with a different job write this exact post? → If YES, it's too generic
2. Does it contain at least one specific detail only this person would know? → If NO, rewrite
3. Would this person say this out loud to a friend? → If NO, it's too corporate
4. Is there something slightly uncomfortable or honest about it? → If NO, it's too safe
5. Would someone save this to read again? → If NO, the value isn't clear enough
