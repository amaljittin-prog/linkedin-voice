## The Psychology Layer

Every great LinkedIn post uses at least one of these 6 triggers:

| Trigger | What It Does | When To Use |
|---------|-------------|-------------|
| **Curiosity Gap** | Opens a loop brain needs to close | Stories, reveals, lessons |
| **Pattern Interrupt** | Breaks prediction, snaps attention | Contrarian takes, single-word openers |
| **Loss Aversion** | FOMO is 2x more powerful than hope | Warnings, "most people don't know" |
| **Vulnerability** | Failure creates deeper connection than success | Personal stories, honest admissions |
| **Social Proof** | People trust what others like them do | Observations, collective experiences |
| **Transformation** | Before → After is neurologically irresistible | Journey posts, lesson posts |
